# **App Name**: StockSim v1

## Core Features:

- User Authentication: Secure registration, login, and password reset functionalities.
- Portfolio Dashboard: Display total portfolio value, available buying power, and daily gains/losses.
- Holdings Details: Show detailed views of owned stocks with shares, cost, market value, and profit/loss.
- Asset Allocation Visualization: Visualize portfolio diversification using a pie chart.
- Stock Search & Trade: Enable searching for stocks by ticker symbol and placing buy/sell orders.
- Watchlist Management: Allow users to create and manage multiple watchlists for tracking stocks.
- Trade History: Track user buy and sell orders, details include shares, price, and realized profit/loss.

## Style Guidelines:

- Primary color: Green (money like) (#08CB00) to evoke trust and stability in trading.
- Background color: black , providing a clean and professional backdrop.
- Accent color: green accents #08CB00 for interactive elements and highlights, ensuring visibility.
- Body and headline font: 'Inter', a sans-serif, for a modern and objective feel.
- Code font: 'Source Code Pro' for displaying any necessary code snippets.
- Use modern icons from Lucide React, maintaining a clean and consistent aesthetic.
- Subtle transitions and animations to provide feedback on user interactions, enhancing usability.
